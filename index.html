    Storm ORM Documentation¶

    Welcome to Storm ORM - a modern C++26 ORM library using compile-time reflection to automatically map C++ structs to database tables without macros.

    Supported backends: SQLite, PostgreSQL

    Quick Links¶

    • Main Project Guide (CLAUDE.md) - Quick start and common tasks
    • Benchmarks - Performance results and benchmark system

    Guide¶

    Documentation for users of Storm ORM — patterns, features, and field/migration reference.

    Cookbook¶

    • Cookbook - Quick-reference patterns for common Storm ORM operations

    Features¶

    • CRUD Operations - INSERT, UPDATE, DELETE with auto-generated IDs
    • SELECT Queries - Optimized row extraction and statement caching
    • WHERE Clauses - Type-safe filtering with pure C++26 reflection
    • JOIN Operations - Single and multi-FK JOINs with type erasure
    • Batch Operations - Bulk INSERT/UPDATE/DELETE with smart thresholds
    • Referential Integrity - Always-on FK constraints, junction FOREIGN KEYs, SQLite PRAGMA
    • Connection Tuning - SQLite busy_timeout default, optional WAL, pooled concurrency

    Reference¶

    • Field Types - Supported C++ to SQLite/PostgreSQL type mappings
    • Migrations - Atlas-based schema migrations with auto-discovered models
