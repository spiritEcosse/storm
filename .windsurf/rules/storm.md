Storm is a lightweight, modular C++23 ORM (Object-Relational Mapping) SQL library designed for seamless database interactions in AI model development. It initially supports SQLite, with a modular architecture to easily integrate other databases (e.g., PostgreSQL, MySQL). Built with C++23 features like concepts, modules, and ranges, Storm ensures modern, efficient, and maintainable code. Google Test is used for robust unit testing. The library prioritizes small, replaceable modules to facilitate swapping database backends while maintaining performance and simplicity.
